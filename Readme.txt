Este programa permite crear una agenda telefónica. Al iniciarlo, aparecerá un menú con cinco opciones: 
1. Guardar contacto, 2. Ver contactos, 3. Eliminar contacto, 4. Editar contacto, 5. Salir. 
La opción deseada se podrá elegir mediante los números del teclado. En caso de elegir un número distinto a 1-5, se escribirá "No ha elegido ninguna opción"

Inicialmente ya hay tres contactos creados con nombre, apellidos y teléfono, todos ellos guardados en una misma variable llamada "contactos".

- En la opción 1 se pedirá el nombre, apellidos y teléfono, separadamente. Una vez escrito, se añadirá a la variable y se escribirán todos los contactos incluyendo el nuevo.
- Si la opción elegida es la 2, se escribirán todos los contactos.
- Con la opción tres se podrá eliminar el contacto elegido. El usuario escribirá el nombre del contacto que desea eliminar, que en caso de no existir se escribirá "Ese contacto no exite". Si el contacto sí existe, se escribirá todos los contactos menos el eliminado.
- Si se elige la opción 4, igual que en la opción 3 se pregunta el nombre del contacto que se quiere editar, que en caso de no existir se escribirá "Ese contacto no exite".
- Se preguntará el nombre, apellidos y teléfono nuevos que se sustituirán en la variable de contactos. Después, se escribirá toda la lista de contactos.
- En la última opción simplemente se escribirá "Ha salido".
